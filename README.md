# API Sistem Pemesanan Tiket Event Online

RESTful API menggunakan Laravel 12 dan MySQL
dengan autentikasi Laravel Sanctum.

## Teknologi
- Laravel 12
- PHP 8.3
- MySQL
- Sanctum
- Postman

## Instalasi
1. git clone repository
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. php artisan serve
